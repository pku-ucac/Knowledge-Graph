习题 1 将分子看作质点之间距离不变的系统，在下列情况下，试求分子的主转动惯量.

a) 分子由位于一条直线上的原子构成.

答案：

$$
I _ { 1 } = I _ { 2 } = \frac { 1 } { \mu } \sum _ { a \ne b } m _ { a } m _ { b } l _ { a b } ^ { 2 } , \qquad I _ { 3 } = 0 ,

$$
